# Pz Gren Kameradentag – Fotogalerie Mockup

Mockup-Seite für die Fotogalerie des Kameradentags der Panzergrenadiere 2026.

## Dateien

```
pzgren-galerie/
├── index.html      ← Hauptseite
├── style.css       ← Alle Styles
├── gallery.js      ← Lightbox-Funktion
└── images/
    ├── img1.jpg
    ├── img2.jpg
    ├── img3.jpg
    ├── img4.jpg
    ├── img5.jpg
    └── logo.png
```

## Auf GitHub Pages hosten

1. Neues Repository auf github.com erstellen (z.B. `pzgren-galerie`)
2. Alle Dateien hochladen (inkl. `images/` Ordner)
3. Gehe zu **Settings → Pages**
4. Unter *Branch* → `main` auswählen → **Save**
5. Nach ca. 1 Minute erreichbar unter:
   `https://DEIN-USERNAME.github.io/pzgren-galerie/`

## Bilder ersetzen

Einfach die Dateien in `/images/` durch eigene Fotos ersetzen –
gleiche Dateinamen behalten (`img1.jpg` bis `img5.jpg`).

Weitere Bilder: im `index.html` einfach neue `.item`-Blöcke im Grid ergänzen.
